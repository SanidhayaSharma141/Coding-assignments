In order to execute the program:
1. Run the program in a Linux system/wsl using gcc compiler.
2. The program takes 2 input n and k from the input.log file available in the folder. 
3. Based on the values, the values from 1 to n will be divided into k threads.
4. After completion, the program will output the perfect numbers found in each process in the OutMain file and individual comparisons in each thread in OutVal file.
